# Weather Diary
**Автор:** Редько Ульяна

## Описание программы
Weather Diary - это графическое приложение для ведения дневника погодных наблюдений. Программа позволяет:

- Добавлять записи о погоде с указанием даты, температуры, описания и наличия осадков
- Фильтровать записи по дате и температуре
- Сохранять все записи в JSON-файл
- Автоматически загружать сохранённые данные при запуске
- Просматривать все записи в удобной таблице

Приложение разработано на Python с использованием библиотеки Tkinter для графического интерфейса.

## Требования
- Python 3.6 или выше
- Стандартные библиотеки Python (tkinter, json, datetime, os) - устанавливать дополнительно не нужно

## Установка и запуск
1. Скачайте файл `weather_diary.py`
2. Откройте терминал (командную строку) в папке с файлом
3. Выполните команду:
   ```bash
   python weather_diary.py
